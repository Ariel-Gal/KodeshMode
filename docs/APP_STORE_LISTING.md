# Garmin Connect IQ Store listing draft

## App name

KodeshMode

## Short description

Hebrew-first Shabbat-aware Garmin watch app with parasha, Hebrew date, and Shabbat times.

## Long description

KodeshMode is a Hebrew-first Garmin Connect IQ watch app designed for users who want a clean Shabbat-aware display on supported Garmin watches.

Features include:

- Hebrew parasha display
- Hebrew date
- Shabbat start and end times
- Shabbat status display
- Optional Shabbat progress
- Battery display
- Configurable clock style, font, size, and time format
- Phone-side settings through Garmin Connect / Connect IQ settings
- Layout controls for display elements

KodeshMode is built as a watch app, not as a watch face, so it can provide dedicated app behavior and Shabbat-mode interaction patterns.

## Permissions explanation

### Positioning

Used to calculate location-based Shabbat times and zmanim.

### Sensor

Used for supported device/app behavior where required by the app.

## Privacy note

KodeshMode is intended to run locally on the watch. Review the source code and Garmin permissions before publishing the final store listing.
